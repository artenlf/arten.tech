O GitHub Blog é uma aplicação em que é possível criar posts no formato de um blog,
utilizando issues de um repositório do GitHub. Por meio da integração à API do GitHub,
são feitas as solicitações e carregadas as issues já formatadas dentro da aplicação.

## Sobre o projeto

Este projeto foi um dos requisitos para conclusão do terceiro módulo na trilha React do Ignite Rocketseat. 
Os objetivos foram: praticar consumo de API (utilizando a API do GitHub), styled-components, roteamento, 
requests utilizando Axios e React Hooks, como o Context/Context-Selector, além de um uso pontual da 
biblioteca react-markdown para exibição e formatação dos posts.

## Features

- **Informações diretamente do GitHub:** Com a integração, todas as informações trazidas à aplicação são as mesmas
que estão nos servidores do GitHub. Incluindo: informações sobre o usuário, data de postagem e comentários em uma issue 
- **Busca dentro do repositório:** É possível fazer buscas por termos especificos dentro de uma issue, utilizando a barra
de pesquisa da aplicação
