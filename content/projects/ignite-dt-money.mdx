O DT Money é uma aplicação simples de controle de finanças, em que é possível adicionar lançamentos de entrada e saída, 
contabilizadas e totalizadas. Também é possível buscar por transações específicas.

## Sobre o projeto

Este projeto foi um dos requisitos para conclusão do segundo módulo na trilha React do Ignite Rocketseat. 
Os objetivos foram criar uma aplicação financeira em que fosse possível criar lançamentos de entrada e saída e exibir os totais.
O projeto também teve como objetivo a utilização de um modal para inserir novas transações e uma barra de buscas para selecionar
transações e valores específicos e também de hooks como o useCallback e useMemo. Entre as tecnologias utilizadas estão o Axios 
para requests e o Zod para validações.

## Features

- **Transações de entrada ou saída:** É possível criar transações de entrada ou saída do fluxo financeiro
- **Busca por transações:** Utilizando a barra de buscas, é possível procurar por transações específicas
- **Totalização:** as transações são totalizadas pelo que é exibido em tela
